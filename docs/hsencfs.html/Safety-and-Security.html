1.4.1 Safety, Security, Feeding and Care

HSENCFS uses BluePoint2 encryption. Bluepoint2 has been thoroughly tested, and withstood the test of time. The backing files in the data directory preserve their original names, size, and access times. The only dependence they need is the original password. This means they can be safely copied from the backing directory for transport (like email / cloud) or backup. Please note that HSENCFS relies on the file system having block size 4096. This is the case with most modern systems, so it does not present an issue. If the block size is not 4096, HSENCFS will operate fine, but the data may not be portable across systems.
